Hardware requirements
====================
- raspberry pi2
    - SD card with raspbian image
- raspberry pi3
    - SD card for Ubuntu core recovery testing
    - USB stick(raspbian image) for Ubuntu core installer
- Dupont cables
    - Pi2 GPIO 2 <-> Pi3 run reset pin
    - Pi2 GND <-> Pi3 GND
- USB to TTL dongle
    - Plug USB dongle to Pi2
    - USB TTL Tx <-> Pi3 Rx
    - USB TTL Rx <-> Pi3 Tx
    - USB TTL GND <-> Pi3 GND
